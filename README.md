# CircSI-SSL-pytorch

- Pytorch implementation of CircSI-SSL: circRNA-binding site identification based on self-supervised learning (CircSI-SSL)



# Usage

- python main.py

- torchvision-0.8.2
- python-3.7.4
- numpy-1.20.2
- scikit-learn-0.19.2
- gensim-3.8.3
- einops-0.5.0
